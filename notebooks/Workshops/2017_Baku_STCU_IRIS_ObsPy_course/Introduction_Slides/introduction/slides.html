What is ObsPy?

Provide complete Python framework to work with all types of seismological data

  • read/write support
    • waveforms (MiniSEED, SAC, GSE, SEISAN, SEG-Y, SEG2, Q, CSS, REFTEK, KNET, ...)
    • station metadata (StationXML, dataless/full SEED, RESP, SACPZ, SC3ML, InventoryXML, Shapefile, ...)
    • event metadata (QuakeML, NDK, NLLOC, ZMAP, CNV, MCHEDR, SC3ML, Shapefile, ...)
      •
  • data center access (FDSN WS, SeedLink, ArcLink, Earthworm, SDS, NRL, Syngine, ...)
  • unified object classes (Waveforms, Events, Stations)
  • signal processing routines
    • trimming, merging, tapering, rotating, resampling, interpolation, ...
    • filtering, instrument correction, ...
    • array analysis, CC routines, PPSD, event detection, TF misfits, ...
      •
